# SwipePaw - React Native Pet Matching App

SwipePaw is a mobile application built with React Native and Expo, designed to pass time in a fun way, but also connect pet owners and animal shelters. 
Users can swipe through pet profiles, express interest, and chat with shelters or other pet owners.

## Features

-   **Pet Swiping**: Browse through profiles of adorable pets and have fun swiping them!
-   **Matching System**: Get notified when there's a mutual interest. (For now, randomly generated)
-   **Shelter Integration**: Discover pets from local shelters.
-   **Conversations**: Chat with shelters or other pet owners to learn more about potential matches. (Coming Soon)
-   **Match Animation**: Engaging animation on successful matches.
-   **Tab Navigation**: Easy navigation using Expo Router's tab layout.

## Technologies Used

-   [React Native](https://reactnative.dev/): A framework for building native mobile apps using JavaScript and React.
-   [Expo](https://expo.dev/): A platform for building, deploying, and quickly iterating on native iOS, Android, and web apps from the same JavaScript/TypeScript codebase.
-   [Expo Router](https://expo.github.io/router/): A file-based routing library for React Native and web apps.
-   [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/): Provides a more comprehensive and flexible approach to creating animations.
-   [React Native Gesture Handler](https://docs.swmansion.com/react-native-gesture-handler/): Provides a robust and efficient way to handle gestures.
-   [@react-native-async-storage/async-storage](https://github.com/react-native-async-storage/async-storage): Community-maintained, AsyncStorage is an unencrypted, asynchronous, persistent, key-value storage system that is global to the app.
-   [Lucide React Native](https://lucide.dev/icons?framework=react-native): Beautifully simple, pixel-perfect icons from Lucide.
-   [@expo-google-fonts](https://github.com/expo/google-fonts): Easy to use Google Fonts with Expo.
-   [Unsplash](https://unsplash.com/): Source of stock photos for pet images.

## Installation

1.  **Clone the repository:**

    ```bash
    git clone <repository-url>
    cd <repository-name>
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Start the development server:**

    ```bash
    npm run dev
    ```
